LocalRun LocalRunSettings

Development mode: Emulate a hadoop run locally on this machine

Production mode: Actually run on hadoop. However, some utilites use local facilities so we still force you to have a local policy.

data PartitionStrategy Source

Constructors

NoPartition

Expect a single key segment emitted from the Mapper.

Partition

Expect a composite key emitted from the Mapper.

Fields

keySegs :: Int

Total segments comprising a key.

partSegs :: Int

First N key segments used for partitioning. All keys that share these segments will be routed to the same reducer.
